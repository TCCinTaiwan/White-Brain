Can't see HTML5 <canvas> element

Here are the most probable reasons why you can't see HTML5 <canvas> element in your browser:

  • You are using an old browser, e.g. Internet Explorer before 9, Firefox before 2.x, Safari before 5.x
  • Internet Explorer 9 or later if there is no <!DOCTYPE html> directive at the top of the HTML page.
  • Internet Explorer 9 or later with compatibility mode turned on.

Note on Internet Explorer compatibility mode

Compatibility mode in Internet Explorer can be turned on/off by the user manually however, sometimes web servers are configured to send the HTTP header X-UA-Compatible: IE=7 which automatically turns the compatibility mode on when a page on this server is opened.
